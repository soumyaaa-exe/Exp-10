Aim
To study and implement the Pandas library in Python for creating and manipulating Series and DataFrames and to perform various data handling operations such as accessing data, modifying data, deleting columns, performing statistical analysis, and filtering data.

Theory
Pandas is an open-source Python library used for data analysis, data manipulation, and data cleaning. It provides powerful data structures that make working with structured data simple and efficient.
The two primary data structures in Pandas are:
1. Series
A Series is a one-dimensional labeled array capable of holding data of any type such as integers, strings, or floating-point numbers.
Example:
Student marks
Temperature readings
Sales values

3. DataFrame
A DataFrame is a two-dimensional table-like structure consisting of rows and columns, similar to a spreadsheet or SQL table.
Each column in a DataFrame can contain different data types.
Pandas provides many useful functions to manipulate data such as:
head() – displays first rows
tail() – displays last rows
shape – returns number of rows and columns
ndim – returns number of dimensions
size – returns total number of elements
columns – displays column names
dtypes – shows data types of columns
DataFrames also support operations like:
Adding new columns
Deleting columns
Updating values
Statistical analysis
Filtering data using conditions
These features make Pandas extremely useful for data science, machine learning, and data analysis tasks.
